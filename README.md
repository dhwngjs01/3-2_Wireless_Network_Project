### 오주헌
## 🔥 <i><u>스마트 화재 경보기</u></i> 🚒🚑

* ### 목적
  화재 발생 시 빠른 대처를 위해 화재 발생 상황을 실시간으로 알려주는 화재 경보기를 제작한다.

* ### 목표
  가스 센서, 온습도 센서, 불꽃 센서로 가스 누출과 화재를 감지한다. 이를 감지하면 LED가 깜빡이고 부저가 울려 화재를 알린다.
  
  또한, 라즈베리파이에 연결된 카메라를 통해 화재 발생 상황을 촬영하여 텔레그램 봇을 통해 사용자에게 전송한다.

* ### 필요 재료
  - [x] 라즈베리파이 (3B+)
  - [x] 아두이노 (UNO)
  - [x] 파이 카메라
  - [x] 온습도 센서 (DHT-11)
  - [ ] 가스 센서 (MQ-2)
  - [ ] 불꽃 센서
  - [ ] LED
  - [ ] 부저
